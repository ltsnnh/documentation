Reset and Clock control
================================================================================

Clock source:

- 8 MHz HSI RC -> SYSCLK
- 4-16 MHz HSE OSC -> SYSCLK (and optional RTCCLK)
- 40 kHz LSI RC -> IWDGCLK (and optional RTCCLK)
- 32.768 kHz LSE crystal -> RTCCLK
